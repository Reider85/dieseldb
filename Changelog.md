Инструкции по запуску
Сначала запустите сервер:

bash
Copy
javac DieselDBServer.java
java DieselDBServer
В другом терминале запустите клиент для тестирования:

bash
Copy
javac DieselDBClient.java
java DieselDBClient
Для теста производительности запустите:

bash
Copy
javac DieselDBPerformanceTest.java
java DieselDBPerformanceTest
Особенности реализации
Сервер поддерживает основные операции CRUD (Create, Read, Update, Delete)

Все данные хранятся в памяти (нет персистентности)

Поддерживаются только строковые значения

Клиент предоставляет удобный API для работы с базой

Тест производительности проверяет:

Вставку 1 миллиона записей

Выборку всех записей

Выборку по условию

Обновление половины записей

Удаление всех записей
